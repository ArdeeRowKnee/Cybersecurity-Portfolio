## Network Layer Communication Analysis
This report comes from the Activity: Analyze network layer communication section of the Google Cybersecurity Professional Certificate. 
In this task I used tcpdump to look at DNS and ICMP traffic and worked out why a website wasn’t loading. 
The packets showed that the DNS service wasn’t responding on port 53. 
It’s a simple scenario but it shows that I can read network traffic, understand DNS behaviour, spot when a service is down, and explain what happened in a clear way.
